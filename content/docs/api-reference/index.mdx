MultiPost 提供的丰富的 API 接口，允许开发者远程调用我们的扩展程序的接口，可以半自动、全自动发布内容到多个平台。

目前，我们提供了 API 接口如下：

## Extension API

在你的 Web App 中，调用 MultiPost Extension 的接口，来实现发布、获取账号信息、获取平台信息等功能。

## Restful API （Beta, Only for Version above 1.1.0）

我们在 multipost.app 中，提供了 Restful API，允许开发者远程调用我们的扩展程序的接口，来实现扩展信息获取、发布任务创建等功能。目前支持远程调用、定时发布等功能。

## Scraper API

我们在 multipost.app 中，提供了 Scraper API，允许开发者远程调用我们的扩展程序的接口，来实现内容爬取、搜索等功能。

## Social Media API

我们在 multipost.app 中，提供了 Social Media API，允许开发者远程调用我们的扩展程序的接口，来实现社交媒体内容爬取、搜索等功能。
