## Overview
Development 专题介绍 MultiPost Extension 的开发流程。帮助你快速参与开发社区，并贡献代码。

## 环境准备
node v20.17.0
pnpm v10.7.0

## 开始使用

首先，安装依赖：

```bash
pnpm i
```

之后，运行 Plasmo 开发服务器：

```bash
pnpm dev
```

在浏览器扩展程序页面中打开开发者模式，点击 `加载已解压的扩展程序` 并找到 `build/chrome-mv3-dev` 进行加载。

## 构建生产版本

运行以下命令：

```bash
pnpm build
```

你可以在 `build` 文件夹下找到构建内容
