### 安装 MultiGet

[Chrome 扩展商店](https://chromewebstore.google.com/detail/multiget/bhggbcoodmbomiaffjbpkafajbgcchgi)

## Features
### Web 保存 
快速保存页面。可以导出 HTML、Markdown 等格式，下载图片、视频。

### 社交媒体保存
社交媒体、如小红书、推特等可保存原格式。
