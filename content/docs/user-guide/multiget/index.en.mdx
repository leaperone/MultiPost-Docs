> Open beta testing in progress, frequent iterations, feedback welcome.

### Install MultiGet

[Chrome Web Store](https://chromewebstore.google.com/detail/multiget/bhggbcoodmbomiaffjbpkafajbgcchgi)

## Features
### Web Saving 
Quickly save pages. Can export HTML, Markdown and other formats, download images and videos.

### Social Media Saving
Social media platforms like Xiaohongshu, Twitter etc. can save in original format.
