## 介绍

MultiPost Analytics 是 MultiPost 推出的一站式分析服务，可以监控你的网页浏览、社交媒体账号等等。  
目前 Web trace 已经上线公测，你可以使用它来分析你自己的网页流量。

它的使用十分简单，你只需要在 MultiPost Web 中，添加你的网站，并将监控代码添加到你的网页 `<head>` 中。

## 使用 
1. 前往 [MultiPost Analytics 仪表盘](https://multipost.app/dashboard/analytics);
2. 在页面右上角添加你的网站;
3. 将监控代码添加到你的网页 `<head>` 中;
4. 刷新你的网页，稍等片刻，你就可以在 MultiPost Analytics 仪表盘中看到你的网站流量了。

## 监控代码

```html
<script
  async
  src="https://multipost.app/tracker/index.js"
  mdata-website-id="<你的网站 ID>"
  mdata-auto-track="true"
/>
```

说明：
- `mdata-auto-track` 是是否自动跟踪你的网页浏览。
- `mdata-website-id` 是你在 MultiPost Analytics 中添加的网站 ID。

## 仪表盘

仪表盘中，你可以看到你的网站流量等等数据，由于目前 Web trace 还在公测中，仍有许多占位符，请注意识别。
