## Scraper 是什么

Scraper 是 MultiPost 的采集工具。主要服务有自动化运营自媒体的用户。

[戳我体验](https://multipost.app/dashboard/scraper) | [查看 API 文档](https://api-docs.multipost.app/6485525m0)

## 主要功能
### 一、通过 API 获取 
1. 网页内容（Markdown、JSON、HTML）；
2. 搜索引擎搜索结果；
3. 社交媒体内容、数据；

### 二、后处理 (WIP)
1. 通过 AI prompt 精准采集网页内容；
2. 通过 AI 搜索引擎搜索结果；

## 使用场景
1. 自媒体运营者：
    - 通过 API 定期采集资讯网站内容，通过 AI Agent 撰写内容，通过 MultiPost 发布内容；
    - 通过 API 获取社交媒体内容、数据，通过 AI Agent 分析热点；
2. 跨境电商：通过 API 获取跨境电商网站热点；
3. 普通人：通过 API 搭建自己的信息流；
