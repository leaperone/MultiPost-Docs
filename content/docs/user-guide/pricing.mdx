## FAQ
### 1. MultiPost 浏览器扩展要收费吗？
> 答：MultiPost 浏览器扩展是免费的。

### 2. 余额可以用于哪些项目？
> 答：用于 MultiPost 的增值业务，如 Restful API，账号监控，采集 API 等等。

### 3. 余额可以退款吗？
> 答：暂不支持退款和提现。

### 4. 充值后多久到账？
> 答：充值后立即到账。

### 5. 如何查看用量？
> 答：[用量统计](https://multipage.app/dashboard/settings/credit-and-usage)
