## 安装 MultiPost 扩展

### 在 Chrome 浏览器中安装

1. 打开 Chrome 浏览器，访问 [Chrome Web Store](https://chromewebstore.google.com/detail/multipost/dhohkaclnjgcikfoaacfgijgjgceofih)。
2. 点击 `添加到 Chrome` 按钮。
3. 安装完成后，可以在浏览器工具栏看到 MultiPost 的图标。

### 在 Edge 浏览器中安装

1. 打开 Edge 浏览器，访问 [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/multipost/ckoiphiceimehjkolnfffgbmihoppgjg)。
2. 点击 `获取` 按钮。
3. 安装完成后，可以在浏览器工具栏看到 MultiPost 的图标。

## 开始使用

在浏览器中，你可以直接使用 MultiPost 的扩展，也可以在 [multipost.app](https://multipost.app) 中创建和发布内容到多个平台。

## FAQ
### 1. 为什么我的网页端正常发布内容，但是扩展不能正常发布？
> 请更新扩展到最新版本。

### 2. 没有我需要的平台怎么办？
> - 请自行开发, 我们提供详细的 API 文档和示例代码。参见 [开始开发](/docs/development)
> - 请在 [GitHub Issues](https://github.com/MultiPost-Extension/MultiPost-Extension/issues) 中提出，等待社区开发
> - 付费定制开发。请联系 [support@leaper.one](mailto:support@leaper.one)

### 3. 为什么在扩展中点击发布后，打开了目标平台，但是没有填写内容？
> 1. 重试；
> 2. 检查扩展版本，更新最新版本；
> 3. 检查内存情况，扩展运行需要充足内存；
> 4. 不要提问类似“为什么我发布不了”、“为什么我发布的内容没有显示”等问题，我们不提供此类问题解答。提问前请阅读[《提问的智慧》](https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way/blob/main/README-zh_CN.md)，并提供详细的环境、版本以及错误信息；
> 5. 如果以上问题都解决了，请在 [GitHub Issues](https://github.com/MultiPost-Extension/MultiPost-Extension/issues) 中提出，等待社区反馈。

### 4. 为什么我发布的内容没有显示？
> 检查发布内容是否符合目标平台的规定；
> 重试、重试、重试，重要的事情说三遍；
> 如果以上问题都解决了，请在 [GitHub Issues](https://github.com/MultiPost-Extension/MultiPost-Extension/issues) 中提出，等待社区反馈。
### 5. 发布视频，点击发布之后，卡住，很久之后只能打开目标平台的网页，为什么？
> 扩展要把视频读到浏览器里，你勾了几个平台，就要读几份视频。内存没空间了，就会向硬盘要虚拟内存。然后磁盘占用率拉满，浏览器就卡死了。
> 解决办法：
> 1. 减少平台，只勾一个平台；
> 2. 压缩视频，控制视频大小；
